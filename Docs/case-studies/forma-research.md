# Forma Shop — Research

## Persona

**Jordan** — 32, urban professional furnishing a new apartment; values quality materials and frictionless mobile checkout.

## User flows

1. **Browse/filter** — Land on shop → filter by category → sort by price → quick-view → open PDP.
2. **PDP purchase** — Gallery → size guide → read reviews → sticky add-to-cart → cart drawer.
3. **Guest checkout** — Cart → shipping (no account) → mock payment → order confirmation.

## Funnel

```
PLP → PDP → Cart drawer → Checkout (3 steps) → Confirmation
         ↑ quick view shortcut
```

## Competitive audit

| Store | Strength | Forma focus |
|-------|----------|-------------|
| Article | Clean PDP, trust | Stronger mobile sticky ATC |
| West Elm | Premium imagery | Simpler guest checkout demo |
