# Habit App — Research & IA

## Flows

1. **Onboarding** — 3 slides → skip or get started → persisted skip flag
2. **Daily check-in** — Home cards → tap complete → streak updates + snackbar
3. **Settings** — Toggles, theme, return to portfolio hub

## Sitemap (bottom nav)

- Home — today's habits + streaks
- Habits — full habit list
- Stats — weekly completion chart
- Settings — profile, notifications, hub link
