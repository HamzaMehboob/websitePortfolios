# Forma Shop — IA & Wireframes

## Routes

```
/shop              Product listing (filters, sort, quick view)
/product/[slug]    PDP (gallery, size guide, reviews, sticky ATC)
/checkout            Multi-step guest checkout
/checkout/confirmation
```

## Checkout steps

1. **Shipping** — name, address, email (guest)
2. **Payment** — card fields (mock) + trust badges
3. **Review** — order summary → place order

## Mobile

- PLP: 2-col grid
- PDP: sticky bottom ATC bar
- Cart: full-width drawer
- Checkout: single column, progress bar top
