# Hub — Problem Statement

## Personas

**Recruiter** — Needs to scan breadth of skills quickly and open live demos without repo archaeology.

**Hiring manager** — Wants evidence of product thinking via case studies, not screenshot galleries.

**Potential client** — Evaluates conversion, accessibility, and brand craft through focused demos.

## Jobs to be done

1. Discover all portfolio demos from one index page.
2. Read why each demo exists (case study narrative).
3. Return to the hub from any demo without getting lost.
