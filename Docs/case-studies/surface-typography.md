# Surface Mag — Editorial Typography

## Type pairing

| Role | Family | Use |
|------|--------|-----|
| Display / H1–H3 | Source Serif 4 | Headlines, article titles |
| UI / meta | IBM Plex Sans | Nav, tags, captions |

## Scale & line length

- Body: 1rem / line-height 1.7, max-width **68ch**
- H1: 2.5–3rem / 1.15
- H2: 1.5rem / 1.25
- H3: 1.25rem / 1.3

## Color

- Ink `#1a1a1a` on paper `#faf9f7`
- Accent `#c45d3a` for links and highlights
- Stone-600/500 for secondary meta

Shared gray ramp aligns with monorepo `ui-tokens` neutral scale.
