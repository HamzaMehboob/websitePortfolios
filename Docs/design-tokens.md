# Design Tokens Guidelines

Per-site brand identity with shared structural tokens from `@website-portfolios/ui-tokens`.

## Shared vs per-site

| Token layer | Shared (`ui-tokens`) | Per-site |
|-------------|----------------------|----------|
| Spacing scale (4/8px) | Yes | — |
| Neutral grays | Yes | — |
| Motion duration / easing | Yes | — |
| Brand primary / accent | — | Yes |
| Typography families | — | Yes |
